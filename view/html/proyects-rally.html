Sistema web para la administración del Rally Networks

ITSOEH
Mixquiahuala de Juárez, Hidalgo

Desarrollo web Back-end PHP MySQL Front-end HTML CSS JQuery AJAX

Resumen

Para este proyecto se utilizaron elementos de la metodología incremental, la persona a cargo de proporcionar los requerimientos fue la Mtra. (pedir permiso) quien se encargó de proporcionar además todos los detalles referentes a la competencia, es importante mencionar que este proyecto es conocido por ser el código base hasta el año 2020, El sistema presentado cuenta con una clara división entre el frontend y backend siguiendo una variación del modelo vista controlador (MVC).



Problema que se resolvió

Crear un sistema web que sirva de apoyo para el control de participantes, calificaciones, distracciones, sanciones, bases y retos.
