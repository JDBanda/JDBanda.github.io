Simulación de un billar

Proyecto escolar, Mixquiahuala de Juárez Hidalgo

Desarrollo de escritorio Java MVC

Resumen

Se realizó un proyecto de escritorio en Java para demostrar que las técnicas de simulación pueden ayudar a comprender como se comporta un negocio a través del tiempo, se analizan específicamente algunas variables aleatorias discretas y continuas. Para la realización de este programa se llevo a cabo el uso de un patrón de diseño basado en MVC.



Problema que se resolvió

Crear una aplicación que aplique técnicas de simulación para representar el comportamiento de un billar


Capturas de pantalla

Vista principal
Se muestra una vista con una única entrada que después de ser ingresada, comenzará a realizar la simulación y arrojará datos en tres tablas como resultado. El proceso emplea una serie de calculos aritméticos para realizar una estimación de los datos.

Gráficos con JFreechart
Los datos de las tablas pueden visualizarce por medios de gráficos realizados en JFreechart 1.5, una librería que permite realizar diferentes tipos de gráficos volviendo esta tarea más sencilla.

Ejemplificaión de MVC
Uno de los propósitos de llevar a cabo esta problemática fue para reconocer como se estructura el patrón MVC de una forma sencilla, gracias a ello los métodos estan claramente delimitaos y el código es más entendible.

Documentación constante
En este programa se pueden observar múltiples comentarios que ayudan a mejorar la comprensión del código fuente.
