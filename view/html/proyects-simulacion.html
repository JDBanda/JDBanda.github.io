Simulación de un billar

Proyecto escolar, Mixquiahuala de Juárez Hidalgo

Desarrollo de escritorio Java MVC

Resumen

Se realizó un proyecto de escritorio en Java para demostrar que las técnicas de simulación puede ayudar a comprender como se comporta un negocio a través del tiempo, se analizan específicamente algunas variables aleatorias discretas y continuas. Para la realización de este programa se llevo a cabo el uso de un patrón de diseño basado en MVC.

Problema que se resolvió

Crear una aplicación que aplique técnicas de simulación para representar el comportamiento de un billar
